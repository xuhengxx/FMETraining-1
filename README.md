# 简介

这是Safe Software的FME Desktop应用程序的高级培训课程手册。

![](.gitbook/assets/img0.000.fmeaboutscreen.png)

此培训建立在基本培训的基础上，涵盖了对希望将技能提升到新水平的所有FME用户都很重要的功能

## 课程结构

全部课程由五个主要部分组成。这些部分是：

* 高级属性处理
* 高级工作空间设计
* 高级读取/写入
* 高级参数使用
* 自定义转换器
